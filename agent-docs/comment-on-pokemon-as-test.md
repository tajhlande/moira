Pokémon is actually a pretty good stress test.

It has:

- large factual space
- many named entities
- hard categorical rules
- interactions between facts
- changing metagames
- authoritative sources

It's much harder than many "research" benchmarks.

A model can bluff its way through:

    What caused the French Revolution?

It has a much harder time bluffing:

    Which OU Pokémon cover Tyranitar's weaknesses?

because the answer depends on many interacting facts.
